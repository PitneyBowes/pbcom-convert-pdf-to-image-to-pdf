Convert PDF-to-Image-to-PDF

Description

This ASP.NET Web Form reads in a PDF file on a given server, converts it to an image, and then converts the image back to a PDF using iTextSharp. It is useful when a browser's internal PDF engine cannot render a PDF correctly. 

Installation

This ASP.NET Web Form has been verified to run on .NET 4.7.2. The iTextSharp library should be placed in the bin/ folder. It assumed the source PDF to convert lives on the same server.
